# Difference between Algorithm and FlowChart:

    Algorithm:
    ----------
    -step by procedure
    -complex to understand
    -text
    -difficult to construct

    Flow chart:
    -----------
    -diagramatic representation
    -easy to understand
    -shape/symbols
    -simple to contruct
